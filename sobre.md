## Para quem é esse curso?

Este minicurso é voltado para alunos que se interessam pelo desenvolvimento Back-end com ferramentas do ecossistema [Spring](https://spring.io), especificamente o **Spring Framework**, mas que ainda não possuem experiência com elas. A proposta desse curso é introduzir ao aluno, de forma didática, quais são os pontos nos quais este framework mais mostra seu valor, além de explorar outras ferramentas da mesma família, como o **Spring Boot** e o **Spring Initializr**, e construir uma boa noção de como desenvolver um sistema Back-end. O conteúdo abordado não busca ensinar o Spring com uma abordagem de baixo-nível, mas sim explicar como as suas diferentes partes se conectam e fornecer um *overview* substancial de suas funcionalidades.

Para este curso, é esperado que o aluno já tenha algum conhecimento da linguagem de programação Java, tenha o mínimo de familiaridade com a linha de comando (o *mínimo* mesmo!) e disposição para aprender novas tecnologias.

Ao final de todas as aulas é esperado que o aluno adquira o conhecimento necessário para *iniciar* sua jornada com esta ferramenta que protagoniza tantas oportunidades no mercado de trabalho, estando preparado para estudos futuros numa maior profundidade, além de entender o que são APIs, como funcionam e como desenvolvê-las.
